# 🎁 AmarelGifts - מערכת בחירת מתנות לעובדים

אתר מודרני לבחירת מתנות לעובדים בחגים, בנוי עם React + Vite ועיצוב בסגנון Apple.

## ✨ תכונות עיקריות

- 🔐 **התחברות בטוחה** - אימות באמצעות תעודת זהות ישראלית
- 🛍️ **קטלוג מתנות** - מגוון רחב של מתנות איכותיות עם סינון לפי קטגוריות
- 🛒 **סל קניות מלא** - ניהול כמויות, הוספה והסרת פריטים
- ✅ **תהליך הזמנה** - סיום הזמנה ללא תשלום עם טופס פרטים
- 🎨 **עיצוב מודרני** - סגנון Apple עם פינות עגולות, ללא הצללות
- 📱 **responsive** - תומך בכל גדלי המסכים

## 🚀 התקנה והפעלה

```bash
# התקנת תלויות
npm install

# הפעלת סביבת הפיתוח
npm run dev

# בניית הפרויקט לפרודקשן
npm run build
```

האתר יהיה זמין בכתובת: `http://localhost:5173`

## 🔐 התחברות

המערכת משתמשת באימות תעודת זהות ישראלית.
דוגמאות לתעודות זהות תקינות:
- `123456782`
- `000000018`

## 🎨 עיצוב וסגנון

הפרויקט משתמש ב:
- **Google Sans** - פונט מודרני שיצא בדצמבר 2025
- **סגנון Apple** - פינות עגולות, עיצוב נקי וצבעוניות רכה
- **ללא הצללות** - עיצוב שטוח ומודרני
